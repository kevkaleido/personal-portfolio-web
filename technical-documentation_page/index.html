Introduction

JavaScript is a versatile, dynamic programming language that runs in the browser as well as on the server. It allows you to create interactive web pages and handle events efficiently.

JavaScript has a rich set of features and a standard library that includes objects like Array, Date, and Math. It can be extended to handle different environments, such as:

  • Client-side JavaScript for creating web pages that interact with the user, manipulate the DOM, and communicate with servers.
  • Server-side JavaScript for creating web servers, interacting with databases, and handling server-side logic.
What you should already know

This guide assumes you have a basic understanding of the following concepts:

  • Basic knowledge of HTML and CSS.
  • Familiarity with general programming concepts like variables, loops, and functions.
  • Experience with web development fundamentals.
JavaScript and Java

Although JavaScript and Java share similar names, they are quite different. JavaScript is a prototype-based, dynamic language, while Java is class-based and statically typed. JavaScript follows Java's naming conventions and basic control-flow constructs but provides more flexibility and simplicity.

In JavaScript, functions can be treated as first-class citizens, while Java enforces strict class structures and type checking.

Hello world

To start writing JavaScript, you can open your browser's console and run the following "Hello, world!" example:

console.log('Hello, world!');

This will display "Hello, world!" in the browser console.

Variables

Variables in JavaScript are used to store data values. You can name variables with letters, underscores (_), or dollar signs ($). JavaScript is case-sensitive, so variable names like myVar and MyVar are different.

Examples of variable names include num, userName, and _temp.

Declaring variables

There are three ways to declare variables in JavaScript:

Using the var keyword (function-scoped):

var message = 'Hello';

Using the let keyword (block-scoped):

let count = 10;

Using the const keyword for constants (block-scoped):

const PI = 3.14;
Variable scope

Variable scope determines the accessibility of variables. JavaScript has function scope and block scope:

Function-scoped variables declared with var:

function showMessage() { var message = 'Hello'; console.log(message); } showMessage(); // Hello

Block-scoped variables declared with let and const:

if (true) { let blockVar = 'Block'; console.log(blockVar); } // Block console.log(blockVar); // ReferenceError
Global variables

Global variables are properties of the global object (window in browsers). They can be accessed from any script in the page:

var globalVar = 'I am global'; console.log(window.globalVar); // I am global
Constants

Constants are declared with the const keyword. Their values cannot be changed once set:

const MAX_USERS = 100;

However, object properties of constants can be modified:

const user = { name: 'John' }; user.name = 'Jane'; console.log(user.name); // Jane
Data types

JavaScript supports several data types, including:

  • Number: let num = 42;
  • String: let str = 'Hello';
  • Boolean: let isTrue = true;
  • Object: let obj = { name: 'Alice' };
  • Array: let arr = [1, 2, 3];
  • Undefined: let undef;
  • Null: let empty = null;
if...else statement

The if statement executes code based on a condition:

if (age > 18) { console.log('Adult'); } else { console.log('Minor'); }

You can also use else if for multiple conditions:

if (score > 90) { console.log('Excellent'); } else if (score > 75) { console.log('Good'); } else { console.log('Needs Improvement'); }
while statement

The while loop executes code as long as the condition is true:

let i = 0; while (i < 5) { console.log(i); i++; }
Function declarations

A function declaration defines a named function:

function sum(a, b) { return a + b; }

Functions can take parameters and return values:

let result = sum(2, 3); console.log(result); // 5
